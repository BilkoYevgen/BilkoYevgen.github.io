About

The Telegram bot is a versatile tool that offers users a range of helpful features to assist with daily tasks. Telegram bot is a powerful tool that offers several features to help users manage their daily routines effectively. The medication reminders, currency exchange rate, task manager, and phonebook features make it a convenient and practical tool for a wide range of users.

My pills

Medication Reminders: The bot allows users to set reminders for taking their medication. The reminders can be scheduled to repeat at a specific interval or set as a one-time notification. This feature is particularly useful for individuals who need to follow a strict medication regimen and may forget to take their medicine on time.

Exchange rate

Currency Exchange Rate: The bot provides the latest official exchange rate of the Ukrainian hryvnia to the US dollar. This feature is helpful for users who need to keep track of currency fluctuations and plan their financial transactions accordingly.

My tasks

Task Manager: The bot includes a task manager that allows users to record and manage their daily tasks. Users can add tasks with specific details and set reminders to complete them at a scheduled time. This feature helps users stay organized and productive throughout the day.

Phonebook

Phonebook: The bot offers a phonebook feature that allows users to store and access contact details. Users can add phone numbers and other information about their contacts, making it easy to find and call them when needed.
